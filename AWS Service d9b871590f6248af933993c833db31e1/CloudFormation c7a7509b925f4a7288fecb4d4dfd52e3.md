# CloudFormation

<aside>
💡 리소스 구성을 템플릿화하여 재사용하기 쉽게 해줌

</aside>

1. Stack
    
    > AWS 리소스의 모음
    > 
    - cloudformation의 템플릿에 의해 생성되고 관리되는 단위
2. Template
    
    > JSON or UAML 포맷의 텍스트 파일
    > 
    - stack을 위한 blueprint 역할
3. Stacksets
    
    > stack의 기능을 확장하여 여러 리전과 여러 계정에서 stack를 생성, 갱신, 삭제하게 해줌
